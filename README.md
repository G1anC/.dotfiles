# My .dotfiles

The Hyprland (ArchLinux) configuration i'm currently using.

### Configurations

```hyprland``` : wayland compositor

```hyprpaper``` : background 

```swaylock``` : lock screen

```Thunar``` : file manager

```VSCode``` : text editor (i'm using 'Kanagawa' as theme)

```wofi``` : applications manager
